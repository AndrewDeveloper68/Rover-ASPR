Для запуска Веб-Сервера нужно находясь в папке control_panel:
1) Ввести в консоль:

```cmd
python -m venv venv
```

2) Ввести в консоль:

```cmd
venv\Scripts\activate.bat
```

3) Ввести в консоль:

```cmd
python -m pip install --upgrade pip
```

4) Ввести в консоль:

```cmd
pip install -r requirements.txt
```

5) Ввести в консоль:

```cmd
python main.py
```

6) Запустить луноход с прошивкой прилагаемой к нему

7) Подключиться к WiFi Лунохода

8) Открыть в браузере http://localhost/
